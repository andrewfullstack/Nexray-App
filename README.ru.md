# Nexray

[English](README.md) · [简体中文](README.zh-CN.md) · [繁體中文](README.zh-TW.md) · [Русский](README.ru.md)

Современный минималистичный VLESS + Trojan + VMess-клиент для macOS, Linux и Windows.
Простой как иконка в трее, быстрый как ядро.

## Первый запуск — обход предупреждений о неподписанном бинарнике

Сборки пока без цифровой подписи. macOS откажется открывать `.dmg`
при первом запуске, а Windows SmartScreen покажет предупреждение перед
запуском `.exe`. И то и другое обходится за секунды:

### macOS — снять флаг карантина

После того как перетащите `Nexray.app` в `/Applications`, выполните в
Терминале (запросит пароль):

```bash
sudo xattr -dr com.apple.quarantine /Applications/Nexray.app
```

Затем открывайте обычным двойным кликом. Если обновляете в тот же
каталог — повторять команду не нужно.

### Windows — Выполнить в любом случае

Дважды кликните по `.exe`. SmartScreen выдаст
**«Система Windows защитила ваш компьютер»**. Нажмите **Подробнее** →
**Выполнить в любом случае**. Предупреждение появляется только при первом
запуске каждой версии.

## Возможности

- **Только современные протоколы.** VLESS (CDN-WS или REALITY), Trojan
  (TCP+TLS) и VMess (TCP+TLS, только AEAD). Отказывается работать с
  Shadowsocks, WebSocket-вариантами vmess/trojan и любыми устаревшими
  или небезопасными по умолчанию комбинациями — это сознательное решение.
- **Импорт подписок** + переключатель «Авто» в каждой группе
  автоматически закрепляет активный сервер за самым быстрым в подписке.
- **Умная маршрутизация.** Встроенные правила `geosite:cn` + ваш
  собственный `rules.conf`. Локальные сайты — напрямую, реклама —
  заблокирована, остальное — через прокси.
- **Системный TUN.** Перехват пакетов на уровне ядра для приложений,
  игнорирующих SOCKS. Корректное восстановление маршрутов при выходе —
  никаких висячих правил.
- **Пять языков интерфейса.** English, 简体中文, 繁體中文, Русский. Язык
  определяется по системному, переключается в Настройках.
- **Минимум сетевой активности.** Никакой телеметрии, никаких
  автообновлений в фоне — пока вы сами не включите Автообновление в
  Настройках.

## Структура репозитория

Этот репозиторий (`Nexray-App`) содержит только публичные релизы и
страницу-витрину — **исходного кода тут нет**.

- Ветка `gh-pages` → рендерится в <https://andrewfullstack.github.io/Nexray-App/>
- Releases → установщики (`.dmg` / `.deb` / `.exe`)

Исходный код ведётся в отдельном приватном репозитории; каждый
релизный тег запускает CI, который собирает установщики и публикует их
(а также этот README и сайт Pages) сюда.
