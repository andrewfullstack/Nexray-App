首次啟動

目前的安裝包未經程式碼簽署。macOS 首次會拒絕開啟 .dmg,Windows 也會以 SmartScreen 攔截 .msi。兩者都可以在數秒內繞過:

macOS — 清除隔離標記

將 Nexray.app 拖進 /Applications 後,在終端機執行一次 (會提示輸入密碼): 

sudo xattr -dr com.apple.quarantine /Applications/Nexray.app

之後正常點兩下開啟即可。覆蓋更新後只要還在同一路徑,無須再次執行。

Windows — 仍要執行

點兩下 .msi,SmartScreen 會顯示 「Windows 已保護您的電腦」。按 其他資訊 → 仍要執行。每個版本只在首次執行時顯示這個警告。

核心特性

僅支援現代協定

VLESS(CDN-WS 或 REALITY)、Trojan(TCP+TLS) 與 VMess(TCP+TLS,僅 AEAD)。拒絕 Shadowsocks、vmess/trojan 的 WebSocket 變體,以及任何過時或預設不安全的協定組合 — 這是設計初衷。

訂閱匯入

貼上任意 Shadowrocket 風格的訂閱連結。伺服器依來源自動分組並測延遲,一鍵匯入。

智慧分流

內建 geosite:cn 規則,加上你自己的 rules.conf。國內站點直連、廣告攔截、其餘流量走代理。

TUN 全域接管

核心級封包擷取,涵蓋不支援 SOCKS 的應用程式。離開時自動還原路由,不留殘餘設定。

自動選最佳伺服器

每個群組的「自動」開關會持續把作用中的設定固定在訂閱中延遲最低的伺服器。

五種語言

English、简体中文、繁體中文、Русский。系統語言自動辨識,亦可在設定中手動切換。
