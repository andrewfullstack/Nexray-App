Первый запуск

Сборки пока без цифровой подписи. macOS откажется открывать .dmg при первом запуске, а Windows SmartScreen покажет предупреждение перед запуском .msi. И то и другое обходится за секунды:

macOS — снять флаг карантина

Перетащите Nexray.app в /Applications, затем выполните в Терминале (запросит пароль): 

sudo xattr -dr com.apple.quarantine /Applications/Nexray.app

После этого открывайте обычным двойным кликом. Если обновляете в тот же каталог — повторять команду не нужно.

Windows — Выполнить в любом случае

Дважды кликните по .msi. SmartScreen выдаст «Система Windows защитила ваш компьютер». Нажмите Подробнее → Выполнить в любом случае. Предупреждение появляется только при первом запуске каждой версии.

Почему Nexray

Только современные протоколы

VLESS (CDN-WS или REALITY), Trojan (TCP+TLS) и VMess (TCP+TLS, только AEAD). Отказывается работать с Shadowsocks, WebSocket-вариантами vmess/trojan и любыми устаревшими или небезопасными по умолчанию комбинациями — это сознательное решение.

Импорт подписок

Вставьте любую подписку в стиле Shadowrocket. Серверы группируются по источнику, латентность измеряется автоматически, импорт — в один клик.

Умная маршрутизация

Встроенные правила geosite:cn + ваш собственный rules.conf. Локальные сайты — напрямую, реклама — заблокирована, остальное — через прокси.

Системный TUN

Перехват пакетов на уровне ядра для приложений, игнорирующих SOCKS. Корректное восстановление маршрутов при выходе — никаких висячих правил.

Авто-выбор лучшего сервера

Переключатель «Авто» в каждой группе автоматически закрепляет активный сервер за самым быстрым в подписке.
